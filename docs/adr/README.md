# Architecture Decision Records (ADR)

Documentación de decisiones arquitectónicas importantes.

## Formato

```markdown
# ADR-XXX: Título de la Decisión

**Fecha:** YYYY-MM-DD
**Estado:** Propuesta | Aceptada | Rechazada | Deprecada

## Contexto
¿Qué problema estamos resolviendo?

## Decisión
¿Qué decidimos hacer?

## Consecuencias
¿Qué impacto tendrá esta decisión?
```

## Índice

- ADR-001: (Pendiente)
